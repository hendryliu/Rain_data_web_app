# Singapore Rainfall Web App — Workflow

## Prerequisites

- Python 3.11+
- [uv](https://docs.astral.sh/uv/) (Python package manager)
- Docker & docker-compose (for deployment)

## Project Structure

```
Rain_data_web_app/
├── data/                        # Raw CSV files (2016–2024, ~7 GB)
├── scripts/
│   └── preprocess.py            # CSV → Parquet + stations.json
├── app/
│   ├── main.py                  # FastAPI server
│   └── static/
│       └── index.html           # Leaflet map + Plotly chart
├── processed/                   # Generated by preprocess.py (git-ignored)
│   ├── stations.json
│   └── rainfall/
│       └── {station_id}.parquet
├── requirements.txt
├── Dockerfile
├── docker-compose.yml
└── .gitignore
```

## Local Development

### 1. Set up the environment

```bash
uv venv .venv
uv pip install -r requirements.txt
```

### 2. Preprocess the data

Converts raw CSVs into per-station Parquet files and a `stations.json` index.

```bash
uv run python scripts/preprocess.py
```

This creates the `processed/` directory with:
- `stations.json` — list of 91 stations with id, name, longitude, latitude
- `rainfall/{station_id}.parquet` — one Parquet file per station with timestamp and reading_value columns

### 3. Start the dev server

```bash
uv run uvicorn app.main:app --reload
```

Open http://localhost:8000 in your browser.

### 4. Use the app

- The map shows all rainfall stations across Singapore
- Click a station marker to see its name and load its rainfall chart
- Use the year dropdown to filter by year (2016–2024) or view all years

## API Endpoints

| Endpoint | Description |
|---|---|
| `GET /` | Serves the frontend (index.html) |
| `GET /api/stations` | Returns all stations as JSON |
| `GET /api/rainfall/{station_id}?year=` | Returns rainfall time series for a station, optionally filtered by year |

## Docker Deployment

> **Note:** Run the preprocessing step locally first — the Docker image copies from `processed/`.

```bash
docker-compose up --build
```

The app will be available at http://localhost:8000.
